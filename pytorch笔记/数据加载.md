## PyTorch数据加载相关语句详细总结

### **一、数据集加载**

#### **1. `torchvision.datasets.CIFAR10()`**

python

```
torchvision.datasets.CIFAR10(root='../data', train=True, transform=torchvision.transforms.ToTensor(), download=True)
```



**作用**：加载CIFAR-10数据集（10类32×32彩色图片）

**参数详解**：

- **`root`**：数据存储的根目录
  - `'../data'`：表示在当前目录的上一级创建data文件夹
- **`train`**：选择数据集类型
  - `True`：加载训练集（50000张）
  - `False`：加载测试集（10000张）
- **`transform`**：数据预处理方法
  - `torchvision.transforms.ToTensor()`：将PIL图像转为Tensor
- **`download`**：是否自动下载
  - `True`：如果本地没有数据就下载
  - `False`：如果本地没有则报错

**返回值**：一个可索引的数据集对象

**使用示例**：

python

```
# 加载训练集和测试集
train_data = torchvision.datasets.CIFAR10(root='./data', train=True, transform=transforms.ToTensor())
test_data = torchvision.datasets.CIFAR10(root='./data', train=False, transform=transforms.ToTensor())

# 获取单个样本
image, label = train_data[0]  # image是3×32×32的Tensor，label是0-9的整数
```

### **二、数据预处理**

#### **2. `torchvision.transforms.ToTensor()`**

python

```
torchvision.transforms.ToTensor()
```



**作用**：将PIL图像或numpy数组转换为PyTorch Tensor

**转换规则**：

- **维度变化**：`(H, W, C)` → `(C, H, W)`
  - H：高度，W：宽度，C：通道数
- **数值变化**：`[0, 255]` → `[0.0, 1.0]`

**使用示例**：

python

```
from PIL import Image
import torchvision.transforms as transforms

# 单独使用
transform = transforms.ToTensor()
img = Image.open('cat.jpg')  # PIL图像，尺寸(32, 32, 3)
tensor_img = transform(img)   # Tensor，尺寸(3, 32, 32)

# 结合Compose使用（常用）
transform = transforms.Compose([
    transforms.ToTensor(),  # 转换为Tensor
    transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))  # 标准化
])
```

### **三、数据加载器**

#### **3. `DataLoader()`**

python

```
from torch.utils.data import DataLoader

train_loader = DataLoader(train_data, batch_size=64, shuffle=True)
```



**作用**：创建可迭代的数据加载器，自动分批打乱数据

**参数详解**：

- **`dataset`**：数据集对象（如CIFAR10返回的对象）
- **`batch_size`**：每批加载的样本数
  - `64`：每次迭代返回64张图片和64个标签
- **`shuffle`**：是否打乱数据
  - `True`：每个epoch重新打乱（训练集用）
  - `False`：顺序读取（测试集用）
- **`num_workers`**：使用几个进程加载数据（默认0，主进程）
- **`drop_last`**：是否丢弃最后不足batch_size的批次

**返回值**：可迭代对象，每次迭代返回一个批次的数据

**使用示例**：

python

```
# 创建加载器
train_loader = DataLoader(
    train_data, 
    batch_size=128, 
    shuffle=True,
    num_workers=4,  # 使用4个进程加速加载
    drop_last=True  # 丢弃最后不完整的批次
)

# 遍历加载器
for batch_idx, (images, labels) in enumerate(train_loader):
    print(f"批次{batch_idx}: 图片尺寸{images.shape}, 标签尺寸{labels.shape}")
    # 输出：批次0: 图片尺寸[128, 3, 32, 32], 标签尺寸[128]
    # [batch_size, channels, height, width]
```

## 一、**内置ImageFolder**

### 方法及功能

| 方法                                       | 功能                                     |
| :----------------------------------------- | :--------------------------------------- |
| `datasets.ImageFolder(root, transform)`    | 创建图像数据集对象，自动按文件夹名分类   |
| `DataLoader(dataset, batch_size, shuffle)` | 创建可迭代的数据加载器，支持批处理和打乱 |

### 使用方法

python

```
from torchvision import datasets
from torch.utils.data import DataLoader

# 加载数据
train_data = datasets.ImageFolder(root='data/train', transform=transform)
test_data = datasets.ImageFolder(root='data/test', transform=transform)

# 创建加载器
train_loader = DataLoader(train_data, batch_size=64, shuffle=True)
test_loader = DataLoader(test_data, batch_size=64, shuffle=False)
```

## 二、**自定义Dataset**

### 必须实现的3个方法

| 方法                       | 功能                                       |
| :------------------------- | :----------------------------------------- |
| `__init__(self, ...)`      | 初始化：定义参数，读取文件列表，准备标签   |
| `__getitem__(self, index)` | 获取单个样本：根据索引返回(数据, 标签)     |
| `__len__(self)`            | 返回数据集大小：让`len(dataset)`能正常工作 |

### 使用方法

python

```
from torch.utils.data import Dataset, DataLoader

class MyDataset(Dataset):
    def __init__(self, root_dir, transform=None):
        self.root_dir = root_dir
        self.transform = transform
        self.images = os.listdir(root_dir)  # 读取所有图片
    
    def __getitem__(self, index):
        img_path = os.path.join(self.root_dir, self.images[index])
        img = Image.open(img_path)
        label = self.get_label(index)  # 自定义标签规则
        return img, label
    
    def __len__(self):
        return len(self.images)

# 使用
dataset = MyDataset(root_dir='data/train', transform=transform)
loader = DataLoader(dataset, batch_size=64, shuffle=True)
```
