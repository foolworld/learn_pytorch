#### 非降维求和

#         ![image-20260214160258355](C:\Users\xs\AppData\Roaming\Typora\typora-user-images\image-20260214160258355.png)

```
# 不加 keepdims (默认 keepdims=False)
sum_A_without = A.sum(axis=1)
print("不加 keepdims 的形状:", sum_A_without.shape)  # torch.Size([5]) - 降维了
print(sum_A_without)
# 输出: tensor([ 6., 22., 38., 54., 70.])

# 加 keepdims=True
sum_A = A.sum(axis=1, keepdims=True)
print("加 keepdims 的形状:", sum_A.shape)  # torch.Size([5, 1]) - 保持2维
print(sum_A)
# 输出:
# tensor([[ 6.],
#         [22.],
#         [38.],
#         [54.],
#         [70.]])
```
