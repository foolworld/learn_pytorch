## FCN知识点

**1. 是什么**

- 全卷积网络，用于图像语义分割

**2. 核心改动**

- 去掉全连接层 → 全是卷积层
- 最后加上上采样层（转置卷积）把图放大回原尺寸

**3. 能干啥**

- 输入一张图，输出一张同样大小的图
- 每个像素点都被分类（比如这个点是猫，那个点是狗）

**4. 和CNN的区别**

| CNN          | FCN            |
| :----------- | :------------- |
| 分类整张图   | 分类每个像素   |
| 输出一个标签 | 输出一张图     |
| 有全连接层   | 无全连接层     |
| 位置信息丢了 | 位置信息保留了 |

**5. 关键操作**

- 卷积层：提取特征（图变小）
- 转置卷积：恢复尺寸（图变大）
- 跳跃连接：融合高低层特征（让分割更精细）

**6. 缺点**

- 分割不够精细（细节容易模糊）
- 对小目标不友好
